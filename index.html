    a simple lisp date-time utility

    i failed to locate an existing utility for dealing quickly with units of time- and date-related information in popular common lisp package repositories. here's the simple way to get my open-source utility for your lisp, via github:

    $ cd ~/quicklisp/local-projects && mkdir cl-date-time
$ cd cl-date-time
$ git clone https://github.com/isismelting/cl-date-time.git

    although i've yet to test this myself, having easy access to the utility on its machine of origin, i assume it should be recognized by quicklisp using (ql:quickload 'cl-date-time) at the repl.

    the project has a lot of room to grow. examine for a moment the expected date-time utilities inside linux, bsd, bare-bones *nix-likes, or even non-ethical operating systems under closed source. it baffles me that we don't demand the same kind of functionality from our lisps. take a look at the source code for this package; toward its end are two string-related functions that represent the first steps toward another dimension of this project.

    the function hastily defun'd as "(time-string)" should end up behaving a lot more like common lisp's famous "(format)" special form, complete with operands to represent fragments of time. the included function is a primitive sketch at best, but it serves well as a placeholder until i figure out how to do this right.
